Software

I predominantly use Open Source software in my life, both at home and at work. It is very rare that I purchase a commercial piece of software, however sometimes there is either no Open Source alternative or the commercial option is outstanding and therefore desirable.
This list of software that I have purchased is as much for my own recollection as for use by others who may be looking for software options.

Purchased Applications

< Software that I need to download and Install >

    Mac OS

    I have a MacBook... I want to run OS X on it!

    Why? Wanted/Needed.

    Mac Operating System >
    BetterZip

    ZIP GUI application for Mac OS X, supports many formats.

    Why? Wanted!

    BetterZip free download >
    Sublime Text

    A truly excellent Text/Code editor that lets you focus on writing. Runs on all platforms!

    Why? Wanted!

    More about Sublime Text>
    Windows 7 and XP

    I have clients who are using them, I run these inside VirtualBoxes on my Mac.

    Why? regrettably needed.

    Windows 7 for Mac >
